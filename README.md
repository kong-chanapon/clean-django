## Run Docker Compose
Navigate to the Project Directory

```bash
  cd clean-django
```

Start the Application
```bash
  docker compose up -d
```

Create a Superuser

```bash
  python3 manage.py createsuperuser
```

Stopping the Application
```bash
  docker-compose down
```
